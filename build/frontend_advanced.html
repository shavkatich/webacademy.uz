Фронтенд-разработчик

Коротко о профессии:

Фронтенд-разработчик - это специалист, создающий те вещи, с которыми будут взаимодействовать люди. Он является ключевой фигурой в процессе создания сайтов, т.к. все этапы разработки проходят вместе с ним. Возможно, именно поэтому это такая забавная работа!

записаться на курс

В курсе "Фронтенд-разработчик, продвинутый" вы освоите:

  • Bootstrap
  • Grunt
  • Less
  • БЭМ

Bootstrap — фреймворк, набор HTML+CSS+JS инструментов и шаблонов для верстки и более эффективного и быстрого создания сайтов и веб-приложений. Включает в себя HTML и CSS-шаблоны оформления для типографики, веб-форм, кнопок, меток, блоков навигации и прочих компонентов веб-интерфейса, включая JavaScript-расширения.

Grunt — это инструмент для сборки javascript проектов из командной строки с использованием задач. Релиз вышел совсем недавно, автор Ben «Cowboy» Alman, проект есть на github.

Less - это CSS-препроцессор, который позволяет использовать переменные, функции, операции и еще много приятного для работы

БЭМ — это методология, которая была создана в Яндексе для разработки сайтов, которые надо делать быстро, а поддерживать долгие годы. Она позволяет создавать расширяемые и повторно используемые компоненты интерфейса.

Программа курса

1. Git console / BEM / Grunt / Less

Урок 1. Управление GIT через консоль.

Базовый рабочий процесс в Гите: фиксация изменений и синхронизация репозиториев. Работа с ветками: создание, синхронизация, слияние. Конфликты и их разрешение. Инструменты для работы с Гитом. Обзор Гитхаба.

Урок 2. БЭМ, теория.

Вёрстка независимыми блоками. Альтернативные методологии. Работа с инспектором. Как верстать, чтобы не ломалось.

Урок 3. БЭМ, практика.

Разметка главной страницы учебного проекта по методологии БЭМ.

Урок 4. CSS-препроцессоры. Часть 1

Вёрстка с препроцессором LESS. Возможности препроцессора: переменные, вложенность правил, импорты, миксины и другие полезные функции. Сборка файлов препроцессора в стилевой файл.

Урок 5. CSS-препроцессоры. Часть 2

Автоматизация сборки препроцессоров. Настройка автоматической компиляцию файлов препроцессора при сохранении.

Урок 6. Адаптивная вёрстка.

Резиновые сетки. Переход от «фикса» к «резине». Принцип «перестройки сетки». «Проблема двух вьюпортов» на мобильных. Мета-тег viewport. Медиавыражения, макро- и микробрейкпоинты. Как организовать код разных версий: мобильной, планшетной, десктопной Единицы измерения vh и vw: примеры использования, проблема «100vw». Адаптивность с фиксированными и резиновыми сетками.

Урок 7. Флексбокс.

Основные понятия: флекс-контейнер и флекс-элемент, главная и поперечная оси. Управление осями, выравнивание и распределение флекс-элементов. Многострочный флекс-контейнер. Управление размерами и «гибкостью» флекс-элементов. Изменение порядка флекс-элементов.

Урок 8. Адаптивная и векторная графика

Экранные и CSS-пиксели. «Ретина» или экраны с повышенной плотностью пикселей. Приёмы ретинизации содержимого веб-страницы. Векторная графика в вебе, формат SVG. Внешний и встроенный SVG, адаптивный SVG, SVG-спрайты. Достоинства и недостатки иконочных шрифтов.

2. Адаптивная верстка / Bootstrap 3

Урок 1. Производительность вёрстки.

Какие CSS-свойства тяжёлые, а какие быстрые. Что нужно знать, когда пользуешься CSS-анимацией и плавными переходами. Что нужно знать о перерисовке (repaint) и перекомпоновке (reflow) страниц. Отличается ли быстродействие сайта на десктопе от мобильного.

Урок 2. Погружение в автоматизацию.

Разберёмся, что лучше для автоматизации Grunt или Gulp. Оптимизация кода и графики. Следуем кодгайду автоматически. Как держать код для разработчика удобным, а для остальных — быстрым.

Урок 3. Знакомство с фремворком Bootstrap.

Что помогает верстать быстрее. Обзор фреймворков, использование на любых проектах. Кросспроектные библиотеки компонентов. Знакомство с фреймворком Bootstrap, подключение фреймворка, базовый шаблон, система сеток, функционирование разметки, создание макета адаптивного сайта при помощи фреймворка Bootstrap, часть 1.

Урок 4. Bootstrap.

Настройки CSS, стилизация HTML-элементов с использованием расширенных классов, и передовая система разметки, типографика glyphicon, выпадающие меню, навигационные панели, сообщения, и многе другое, добавление пройденных инструментов на учебный макет, создание макета адаптивного сайта при помощи фреймворка Bootstrap, часть 2. Выбор собственного макета сайта.

Урок 5. Компоненты JavaScript в Bootstrap.

Изучение готовых компонентов JavaScript, вкладки, модальные окна, подсказки, информеры, кнопки, сворачивающиеся элементы, слайдер/карусель, добавление на текущий макет, создание макета адаптивного сайта при помощи фреймворка Bootstrap, часть 3.

Урок 6. Верстка Landing page на Boostrap. Часть 1.

Урок 7. Верстка Landing page на Boostrap. Часть 2.

+ Предварительная сдача персонального проекта. Работа над ошибками.

Урок 8. Верстка Landing page на Boostrap. Часть 3.

+ Защита персонального проекта.
